# Handiproject

## Install dependencies

    npm install

Le `npm install` est important pour installer toutes les dépendences nécessaire au bon fonctionnement de l'application

## Launch App

Example to run the HandiApp connected to DataBase:

    npm run dev

Aussi simple que ça !
